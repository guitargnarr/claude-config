## Theme Factory

I'll help you work with themes for consistent styling across your outputs. Let me process your theme request.

Available actions:
- **list**: Show all available themes
- **preview [theme]**: Generate visual preview of a theme
- **export [theme] --format [css|json]**: Export theme in specified format
- **apply [theme]**: Apply theme to matplotlib for current session

Available themes:
- anthropic, tech_modern, matthew_custom, dark_mode, cyberpunk
- ocean, professional_light, warm_earth, monochrome, nature

I'll now execute your theme request using the theme_factory module.
