# Common Pitfalls

## Platform-Specific Issues

- ❌ **NO vercel.json for Vite projects** - Vercel auto-detects
- ❌ **NO Tailwind v4** - Use v3 (config incompatibility)
- ❌ **NO extensive planning before coding** - Implement first
- ❌ **NO lock file operations on coordination files** - Use audit-sync carefully
- ✅ **Absolute URLs for OG meta tags** - Relative paths fail in social shares
