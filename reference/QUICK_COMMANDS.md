## Job Search Workflow

**Daily:**
1. Check Robert Half portal for new matches
2. Update Job Search.numbers with applications
3. Run gmail_auto_sync.py to catch confirmations
4. Respond to interview requests immediately

**Weekly:**
1. Prepare unemployment certification (track 5+ activities)
2. Review pending applications for follow-ups
3. Update resume variants if needed
