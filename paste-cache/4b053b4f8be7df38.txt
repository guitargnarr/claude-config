 Frontend Design Oracle: Tenerife PAL Transformation

  Design Philosophy Shift

  Current Problem: The page operates as a warning system when it should be a confidence engine.

  The fundamental UX error: treating uncertainty as danger. A knowledgeable travel companion doesn't say "RISK LEVEL: MEDIUM-HIGH" — they say "Here's exactly what to do, and I've got backup plans ready."

  ---
  Aesthetic Direction: Volcanic Calm

  Tenerife is defined by two forces: the volcanic drama of El Teide and the serene Atlantic. The design should mirror this — grounded confidence with moments of warmth.

  Color Palette Transformation

  OUT: Alarm-based palette
  Red warnings → Orange/amber warmth
  Green/Yellow/Red traffic light → Unified teal progression
  Multiple competing alert colors → Single accent system

  IN: Canary Islands-inspired palette

  [Flight details]

      ↓

  ---
  3. Kill the Traffic Light Scenario Matrix

  The three-column BEST/LIKELY/WORST design is inherently anxiety-inducing because:
  - It gives equal visual weight to failure
  - Traffic light colors trigger stress responses
  - It frames the trip as probability, not preparation

  Replace with: A confident single timeline (shown above) with collapsed edge cases.

  If you must show scenarios, use a progressive disclosure pattern where the default state shows the likely (successful) path, and alternatives are available but not prominent.

  4. Typography & Spacing

  Font Pairing (add to index.html):
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link href="https://fonts.googleapis.com/css2?family=DM+Sans:wght@400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet">

  :root {
    --font-sans: 'DM Sans', system-ui, sans-serif;
    --font-mono: 'JetBrains Mono', monospace;
  }

  DM Sans: Friendly but professional. Not as cold as Inter, more personality than system fonts.
  JetBrains Mono: For times, flight numbers, codes — gives technical info a polished look.

  Spacing principles:
  - More breathing room between sections (py-8 minimum)
  - Tighter internal spacing within cards
  - Consistent 24px (p-6) padding on all cards

  5. Motion That Builds Confidence

  Remove: Jittery hover states, pulsing warnings, attention-grabbing animations

  Add: Smooth, purposeful transitions that feel like pages turning

  // Stagger children on section reveal

  const itemVariants = {

  // Use on flight segments
